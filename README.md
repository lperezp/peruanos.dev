Directorio centralizado de eventos, comunidades y proyectos open source de la comunidad tech peruana.

## 🚀 Características

- **📅 Eventos:** Descubre conferencias, meetups, talleres y webinars (presenciales, virtuales o híbridos) de la comunidad tech.
- **👥 Comunidades:** Encuentra y únete a diversas comunidades tecnológicas en Perú.
- **💻 Proyectos Open Source:** Explora y contribuye a proyectos open source creados por desarrolladores peruanos.

## 🛠️ Tecnologías

Este proyecto está construido con herramientas modernas de desarrollo web:

- [Next.js](https://nextjs.org/) (App Router)
- [React](https://react.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [TypeScript](https://www.typescriptlang.org/)

## 💻 Desarrollo Local

Para ejecutar el proyecto localmente, sigue estos pasos:

1. **Clona el repositorio:**
   ```bash
   git clone https://github.com/lperezp/peruanos.dev.git
   cd peruanos.dev
   ```

2. **Instala las dependencias:**
   ```bash
   npm install
   ```

3. **Inicia el servidor de desarrollo:**
   ```bash
   npm run dev
   ```

4. **Abre tu navegador:**
   Visita [http://localhost:3000](http://localhost:3000) para ver la aplicación en funcionamiento.

¡Todas las contribuciones son bienvenidas! Puedes ayudar agregando nuevos eventos, comunidades, proyectos open source, o mejorando el código.
